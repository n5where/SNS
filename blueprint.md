# 프로젝트 블루프린트: 이재명 대통령 SNS 대시보드

## 개요
이 프로젝트는 이재명 대통령의 공식 소셜 미디어 활동을 한눈에 볼 수 있는 웹 기반 대시보드입니다. 페이스북(Facebook)과 엑스(X, 구 Twitter) 피드를 나란히 배치하여 최신 소식을 실시간으로 제공하는 것을 목표로 합니다.

## 기능 명세
- **통합 뷰**: 화면을 분할하여 좌측(또는 상단)에는 페이스북 피드, 우측(또는 하단)에는 X 피드를 표시합니다.
- **반응형 디자인**: 데스크탑에서는 가로 배치, 모바일에서는 세로 배치로 자동 조정되어 모든 기기에서 최적화된 화면을 제공합니다.
- **실시간 피드**: 각 플랫폼의 공식 위젯을 사용하여 실시간으로 업데이트되는 타임라인을 제공합니다.
- **접근성 준수**: 시맨틱 태그 사용 및 명도 대비 등을 고려하여 모든 사용자가 접근하기 쉽도록 설계합니다.

## 현재 구현 계획
1.  **구조 설계 (`index.html`)**:
    *   헤더: 제목 및 간단한 소개.
    *   메인 콘텐츠: 두 개의 주요 섹션(페이스북, X)을 포함하는 컨테이너.
    *   푸터: 저작권 및 링크.
2.  **스타일링 (`style.css`)**:
    *   Flexbox를 이용한 2단 레이아웃 구현.
    *   모바일 우선(Mobile-First) 미디어 쿼리 적용.
    *   가독성을 높이는 타이포그래피 및 깔끔한 UI 디자인.
3.  **스크립트 (`main.js`)**:
    *   필요 시 위젯 로드 제어 또는 동적 레이아웃 조정 (초기에는 최소화).
4.  **외부 연동**:
    *   Facebook Page Plugin 연동.
    *   X (Twitter) Embedded Timeline 연동.
